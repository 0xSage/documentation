# Faucets

A faucet is a service that provides small amounts of xDai tokens to users who are experimenting with Gnosis. Here is a list of the available faucets.

## Oficial Faucet

- [Gnosis Faucet](https://gnosisfaucet.com/)


## Community Faucets

- [Gimlu](https://www.gimlu.com/faucet)
- [Stakely](https://stakely.io/en/faucet/gnosis-chain-xdai)
- [Unitap](https://unitap.app/gas-faucet)
- [xDai Faucet](https://xdai-faucet.top/)
